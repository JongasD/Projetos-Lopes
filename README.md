## 🌟 Visão Geral

O **Projeto Lopes** é um site institucional feito do zero com **HTML, CSS e JavaScript puro**, pensado para destacar um corretor de imóveis, exibir imóveis em destaque, serviços oferecidos e facilitar o contato com clientes.

Ideal para quem busca uma solução leve, rápida e sem dependências externas.

## ✨ Funcionalidades

✔️ Design moderno e limpo  
✔️ Totalmente responsivo (móvel, tablet, desktop)  
✔️ Seção de imóveis com cards interativos  
✔️ Formulário de contato funcional (via formulário embutido ou integração futura)  
✔️ Menu hambúrguer para mobile  
✔️ Animações suaves com CSS  
✔️ Scroll suave entre seções  
✔️ Feito 100% com tecnologias web nativas
